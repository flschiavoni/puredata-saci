# puredata-saci
Saci is a Pure Data external that mess up your code.

TO COMPILE
==========
make

TO INSTALL
==========
Install locally (Linux)

`mkdir -p ~/pd-externals/saci`
`cp saci.pd_linux ~/pd-externals/saci/.`
`cp saci_help.pd  ~/pd-externals/saci/.`
`cp saci-meta.pd  ~/pd-externals/saci/.`

Have fun!
